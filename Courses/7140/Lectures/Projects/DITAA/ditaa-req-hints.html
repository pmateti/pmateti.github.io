3 Build Tutorial

  1. Your Final Project Document must include a build tutorial.
  2. In terms of SE life-cycle, this is not part of requirements. But, we are working with an existing product.
  3. We should build ditaaXYZ.jar (version number XYZ included) and ditaa-eps.jar. Usage: java -jar ditaa-eps.jar input.txt output.eps

4 Kotlin?

  1. Preferably use Kotlin instead of Java. But, you are not forced to.
    1. JetBrains IDEA can auto-convert Java to Kotlin.
  2. https://fabiomsr.github.io/from-java-to-kotlin/ Quick comparison.
  3. IntelliJ IDEA http://kotlinlang.org/docs/tutorials/getting-started.html
  4. The Kotlin plugin is bundled with IntelliJ IDEA and is activated by default.
