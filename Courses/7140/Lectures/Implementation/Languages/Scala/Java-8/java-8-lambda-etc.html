3.1 Functions without Names (Lambda syntax)

  1. Suppose we have f(x, y) defined as (x + 1)*(y + 2). So, f(3, 4)==24.
  2. The anonymous version of f is: lambda x, y: (x + 1)*(y + 2) ;
  3. The f(3, 4) is written as (lambda x, y: (x + 1)*(y + 2))(3, 4)
  4. Function definition appears in-line with function-invocation.
  5. Another, but same, in-line def makes it another anonymous function that we happen to know is the same.

4 Java8 Functions without Names

  1. (int x, int y) -> { return (x + 1)*(y + 2); }
  2. (x, y) -> { (x + 1)*(y + 2); } as above but with inferred types
  3. x -> { System.out.println(x); } is an example of single parameter with inferred type on the left of the arrow, and a block with no return value on the rhs.
  4. Without Lambda exp 
    Runnable r1 = new Runnable() {
  public void run() {
    System.out.println("Running without Lambda");
  }
};

    With Lambda exp 

    Runnable r2 = () -> { System.out.println("Running with Lambda"); };

    No inputs.

  5. "burger arrow" 
    for(String s: getACollection()) {  // simple loop ...
}

for(String s: () -> getAnIterator()) {  // not a simple loop ...
}
  6. Selecting persons based on some criteria 
    printPersons(
    roster, (Person p) ->
        p.getGender() == Person.Sex.MALE && p.getAge() >= 18
);
