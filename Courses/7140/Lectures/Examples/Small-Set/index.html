1 Abstract

This subtree of notes describes the requirements, specs, design and implementation of a SmallSet of integers. It is similar, but not equivalent to Java Set. Our description is focused on abstraction and assertions. This is expected to be a good example of software engineering "in the tiny". But, bug-free?!

It is also intended as an example for use later with learning JML assertions, JUnit/TestNG, and FindBugs.

This is NOT an implementation of Java Set container.
