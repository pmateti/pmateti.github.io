Home work in this course is done individually, without help from others. The homework will be given piece meal and it is expected that you will work on them as and when given without waiting for a due date. The due date is announced on the course home page.

This homework is optional. Points earned on this homework are considered "bonus points." I explained in the lectures how I intend to use these points. Here is a summary of that explanation. I determine the grading "curves" without using the bonus points. I then add the 10% weighted bonus points to your total score. This may bring you up (at most) by one letter-grade (e.g., from a B to an A).

  2. (20 points) [JML] Thoroughly document the Java program selected for the problem above using JML assertions (i.e., invariant, requires, ensures, assignable, ...).
