2.2 Exercises

  1. Non-Functional Requirement: Assignments to array elements are disallowed. Instead, you are given a primitive exchange(i, j) that exchanges a[i] with a[j].
  2. Design and implement a Quicksort program that honors the above. You are welcome to start with the Java program from the Princeton book. [We are skipping requirements and specs.]
  3. Specify exchange(i, j).
  4. Should we revise exchange(i, j) to exchangea(a, i, j)? Context: Design of API.
  5. Do an "intriguing" implementation of exchange(i, j). What do you mean "intriguing" ?!
