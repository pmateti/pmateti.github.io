This is a closed book, closed notes exam, except for the few items listed in the last class. Do not give or take help during the exam.
  1. (6*5 points) The following statements may or may not be (fully or partially) valid.   Explain/ discuss/ dispute the statement.  It is possible to write no more than, say, ten, sentences each, and yet receive full score.
    1. There are examples of safety properties that are liveness properties.
    2. Every distributed system must also be a concurrent system.
    3. Asynchronous  message passing is more fundamental than synchronous message passing.
    4. Token-passing algorithms were discussed under the heading of ``asynchronous message passing'' because token-passing does not make sense under synchronous message passing.
    5. The happened before relation, as in the book, requires a logical clock for every process.  But  a typical machine (node) in a distributed system runs several processes. It is sufficient to maintain a logical clock per node that is shared by all processes running on that node.
    6. The weakly fair scheduling policy is impractical.
  2. (5+5+10 points) It is possible to write no more than, say, ten, sentences each, and yet receive full score.
    1. Describe the problem of "termination detection."
    2. Detection of termination is difficult only in peer-to-peer distributed computing not in client-server computing.
    3. Describe a termination detection algorithm of your choice.
  3. (50 points) You are given a large (in the zillions) bag B of triplets of unsigned integers (x, y, d).  Each triplet represents a horizontal line segment of length d in the Euclidean plane whose left end point is at (x, y).   A (x1, y1, d1) contains (x2, y2, d2) if y1 == y2 and x1 <= x2 and x1 + d1 >= x2 + d2.  We wish to minimize the size (== number of triplets) of  the given bag of triplets so that every original triplet is contained in some triplet in the resulting collection. 
    1. As usual, we wish to maximize concurrency, we prefer a symmetric solution, we rate correctness much higher than efficiency, and cannot tolerate deadlocks or livelocks. Defend why we prefer a symmetric solution (5 points) , and  why we rate correctness much higher than efficiency (5 points) .
    2. Fully explain your solution design (10 points) in English for Linda and implement  (10 points) it in C-Linda. Assume that the tuple space already contains the triplets <"t", x, y, d> of B, and the size of this bag <"sz", N>. You lose 2 points for each use of inp or rdp.
    3. (c) Fully explain your solution design (10 points) in English for CSP.   It is not required that this use the same "design idea(s)" of the above. Implement  (10 points) the design in CSP.  The bag is generated by process Q whose structure is Q :: *[ ... --> P!(x, y, d) ].  Develop the process P and any others.
