  1. "Address space layout randomization (ASLR) is a memory-protection process for operating systems (OSes) that guards against buffer-overflow attacks by randomizing the location where system executables are loaded into memory."

  3. Several mainstream, general-purpose operating systems implement ASLR. Android, BSDs, Linux, iOS, Windows.
