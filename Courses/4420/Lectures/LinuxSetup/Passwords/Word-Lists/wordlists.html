Password Cracking Word Lists

  1. The following are readily available "word lists" in the context of choosing good passwords. This is obviously not an exhaustive list. Note also that some files are populated at multiple sites.
  2. /usr/share/wordlists/* of the Kali Linux distribution.
  3. ./500-worst-passwords.txt
  4. Past collections of broken password databases
  5. https://github.com/danielmiessler/SecLists/ is the security tester's companion. It is a collection of multiple types of lists used during security assessments. List types include usernames, passwords, URLs, sensitive data grep strings, fuzzing payloads, and many more. It includes a collection of password lists. It includes the RockYou lists.
  6. http://contest-2010.korelogic.com/wordlists.html wordlists used in "Crack Me If You Can" contest of DEFCON 2010.
  7. https://wiki.skullsecurity.org/Passwords Passwords that were leaked or stolen from sites.
  8. http://gdataonline.com/downloads/
  9. http://www.justpain.com/ut_maps/wordlists/
  10. http://weakpass.com/lists
  11. http://www.adeptus-mechanicus.com/codex/hashpass/hashpass.php
  12. https://github.com/danielmiessler/SecLists/tree/master/Passwords is a collection of multiple types of lists used during security assessments. List types include usernames, passwords, URLs, sensitive data grep strings, fuzzing payloads, and many more. It includes a collection of password lists. It includes the RockYou lists.
  13. http://www.openwall.com/wordlists/
  14. Mark Burnett, Today I Am Releasing Ten Million Passwords, Feb 9, 2015.
  15. /usr/share/wordlists/* of the Kali Linux distribution.
